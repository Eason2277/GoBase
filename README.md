# GoBase

一个用于系统学习 Go 语言基础与后端开发的练习仓库。

本仓库以小型、可独立运行的示例为主。每个示例集中讲解一个知识点，方便理解 Go 的语法、运行机制和设计思路。

## 学习目标

- 掌握变量、常量、类型、流程控制和函数等基础语法
- 理解数组、切片、map、结构体和指针
- 学习方法、接口及 Go 的错误处理方式
- 理解 goroutine、channel、context 和并发安全
- 熟悉 Go Modules、测试、格式化和常用工具链
- 逐步实践 HTTP、数据库、缓存和鉴权等后端知识

## 当前内容

| 文件 | 内容 |
| --- | --- |
| `main.go` | 输出 `Hello World` 的第一个 Go 程序 |
| `variable01.go` | 变量声明、类型推断、重新赋值、零值和多变量声明 |

## 环境要求

- macOS 或其他支持 Go 的操作系统
- 已安装 Go

查看当前 Go 版本：

```bash
go version
```

本仓库当前使用 Go 1.26.5 进行练习和验证。基础示例通常也能在较早的现代 Go 版本中运行。

## 运行示例

克隆仓库：

```bash
git clone https://github.com/Eason2277/GoBase.git
cd GoBase
```

运行 Hello World：

```bash
go run main.go
```

预期输出：

```text
Hello World
```

运行变量示例：

```bash
go run variable01.go
```

当前多个示例文件都属于 `main` 包，并且各自包含 `main` 函数，因此需要单独运行指定文件。这里不使用通常用于运行整个包的 `go run .`；后续会随着 Go Modules 和包结构的学习重新整理目录。

## 学习路线

```text
基础语法
  -> 复合数据类型
  -> 函数与错误处理
  -> 方法与接口
  -> 并发编程
  -> HTTP 服务
  -> 数据库与缓存
  -> 完整后端项目
```

## 练习方式

学习每个示例时，可以按照以下顺序进行：

1. 阅读代码并预测输出。
2. 使用 `go run 文件名.go` 运行程序。
3. 修改变量或逻辑，观察结果变化。
4. 尝试解释编译错误产生的原因。
5. 使用 `gofmt -w 文件名.go` 格式化代码。

## 代码约定

- Go 代码使用 `gofmt` 格式化
- 代码注释使用简洁、自然的英语
- 示例保持小而清晰，避免引入不必要的第三方依赖
- 每次练习尽量只聚焦一个核心知识点
